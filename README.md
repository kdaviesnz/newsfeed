# newsfeed

## Install

Via Composer

``` bash
$ composer require kdaviesnz/newsfeed
```

## Usage

``` php

        $newsfeed = new \kdaviesnz\newsfeed\Newsfeed();
        var_dump(json_decode($newsfeed));

```
